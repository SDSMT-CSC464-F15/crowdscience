\subsection{\-Detailed \-Description}
\-Contains a simplified map leaflet for the purpose of reporting an event. \-This file contains functions specific to the map leaflet used on the report screen\-: \hyperlink{report_map_leaflet_8js_abcf96615dae15880bbc5e29ee0395df8}{initmap()}, \hyperlink{report_map_leaflet_8js_a7f405e083733a4bf6448065159675dd2}{add\-Marker()}, and \hyperlink{report_map_leaflet_8js_a2e51fc054b9efb976d592be0293e94ec}{on\-Map\-Click()}. \-The \hyperlink{report_map_leaflet_8js_abcf96615dae15880bbc5e29ee0395df8}{initmap()} function creates a new instance of the map leaflet, setting parameters for a simplified display. \-The \hyperlink{report_map_leaflet_8js_a7f405e083733a4bf6448065159675dd2}{add\-Marker()} function adds a marker to the map. \-The \hyperlink{report_map_leaflet_8js_a2e51fc054b9efb976d592be0293e94ec}{on\-Map\-Click()} function defines what is done when the user clicks on the map, and adds the latitude and longitude of the marker to the text boxes associated with the map on the report screen.

\-The map, popup, and marker global variables store a reference to the map and all its markers being displayed.
