\subsection{\-Detailed \-Description}
\-Contains functions for checking if a user is logged in. \-This file contains {\ttfamily }.ready , which sets the function to be called when the document, the \-H\-T\-M\-L loading the \-Java\-Script, has been loaded and is ready to run; and two functions\-: {\ttfamily check\-Login} and {\ttfamily update\-User\-Status} . \-The {\ttfamily }.ready function calls the check\-Login function and then generates necessary event handlers. \-The {\ttfamily check\-Login} posts a request to {\ttfamily \hyperlink{check_login_8php}{check\-Login.\-php}} to see if a user is saved in the session information, when the post completes successfully, the {\ttfamily update\-User\-Status} function is called. \-The {\ttfamily update\-User\-Status} function updates the \-H\-T\-M\-L according to whether a user is logged in.
