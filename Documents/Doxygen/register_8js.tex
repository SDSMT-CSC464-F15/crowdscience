\-Clears error messages and then generates necessary event handlers. 

\-This function hides the error messages indicating mismatched passwords and or an already taken username, and then sets the event handler for the register form submission event. \-The event handler will clear old error messages, and if the form passwords match, will call the \hyperlink{register_8js_ac86cbba17a6824ce637636999566121e}{\-P\-O\-S\-T\-\_\-\-Register\-User()} function to register the user. \-If the passwords don't match, the fields are reset, and an error message displayed to the user.
